ARCHITECTURE TECHNIQUE
================================================================

Stack :
  - Frontend : React 19 + Vite 7 + TypeScript
  - Routeur  : wouter (SPA, pas de react-router)
  - Backend  : Supabase uniquement (PostgreSQL + Auth + Storage)
  - Deploy   : Vercel — vercel.json :
      installCommand : pnpm install
      buildCommand   : vite build
      outputDirectory: client/dist
      rewrites       : [{ source: "/((?!assets/).*)", destination: "/index.html" }]
  - PDF      : jsPDF + jspdf-autotable
  - Package  : pnpm (pnpm-lock.yaml présent)

Chemins clés :

Tables Supabase :
  profiles  : id, email, role, first_name, last_name, phone,
              adresse_facturation, ville_facturation, cp_facturation, pays_facturation,
              adresse_livraison, ville_livraison, cp_livraison, pays_livraison,
              adresse_livraison_identique (boolean)
  quotes    : id, user_id, numero_devis, produits (JSONB), prix_total_calcule,
              prix_negocie, statut, facture_generee, signe_le,
              adresse_client, ville_client, pdf_url, facture_url,
              acompte_pourcentage, acompte_paye, solde_paye,
              date_livraison_prevue, notes_internes, bl_genere, created_at
  partners  : id, user_id, nom, email, telephone, actif, created_at
  products  : id, nom, reference, reference_interne, categorie, description,
              prix_achat, prix_public, image_url, actif, ordre, updated_at
  invoices  : id, numero_facture, quote_id, user_id, date_facture,
              montant_ht, montant_acompte, type_facture, statut, pdf_url (V5)
  delivery_notes : id, numero_bl, quote_id, user_id, date_livraison,
              adresse_livraison, produits (JSONB), statut, pdf_url (V5)
  fees      : id, numero_document, quote_id, type_frais (maritime/dedouanement),
              montant_ht, description, details (JSONB), pdf_url (V5)
  commission_notes : id, numero_commission, partner_id, quote_id,
              montant_commission, statut, pdf_url (V5)
  admin_params : key, value (JSONB), label — émetteur, rib, acompte_defaut (V5)
  site_content : CMS (logo, navigation, bannière, config pages)

Rôles :
  visitor | user | vip | partner | collaborateur | admin
  Source unique : client/src/utils/calculPrix.ts → calculerPrix()
  Multiplicateurs : user\u00D72, partner\u00D71.2, vip\u00D71.3
