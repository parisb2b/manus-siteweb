ARCHITECTURE TECHNIQUE
================================================================

Stack :
  - Frontend : React 19 + Vite 7 + TypeScript
  - Routeur  : wouter (SPA, pas de react-router)
  - Backend  : Supabase uniquement (PostgreSQL + Auth + Storage)
  - Deploy   : Vercel — vercel.json :
      installCommand : pnpm install
      buildCommand   : vite build
      outputDirectory: client/dist
      rewrites       : [{ source: "/((?!assets/).*)", destination: "/index.html" }]
  - PDF      : jsPDF + jspdf-autotable
  - Package  : pnpm (pnpm-lock.yaml présent)

Chemins clés :
  /Users/mactell/manus siteweb/           ← racine du projet
  client/src/App.tsx                      ← toutes les routes
  client/src/components/Header.tsx        ← navigation + logo
  client/src/pages/MonCompte.tsx          ← espace client complet
  client/src/pages/Cart.tsx               ← panier
  client/src/pages/admin/AdminLayout.tsx  ← back-office
  client/src/contexts/AuthContext.tsx     ← auth + profil
  client/src/contexts/CartContext.tsx     ← état du panier
  client/src/hooks/useRole.ts             ← rôle utilisateur
  client/src/utils/calculPrix.ts          ← source unique des prix
  client/src/features/pdf/               ← moteur PDF (v3.0)
    lib/pdf-theme.ts                      ← constantes couleurs
    lib/pdf-helpers.ts                    ← formatPrix, formatDate, EMETTEUR
    lib/pdf-engine.ts                     ← fonctions partagées
    templates/quote-pdf.ts               ← devis
    templates/invoice-pdf.ts             ← facture
    templates/commission-pdf.ts          ← commission
  client/src/utils/generateDevisPDF.ts    ← shim 7 lignes → features/pdf
  client/src/utils/generateFacturePDF.ts  ← shim 7 lignes → features/pdf
  client/src/utils/generateCommissionPDF.ts ← shim 5 lignes → features/pdf

Tables Supabase :
  profiles  : id, email, role, first_name, last_name, phone,
              adresse_facturation, ville_facturation, cp_facturation, pays_facturation,
              adresse_livraison, ville_livraison, cp_livraison, pays_livraison,
              adresse_livraison_identique (boolean)
  quotes    : id, user_id, numero_devis, produits (JSONB), prix_total_calcule,
              prix_negocie, statut, facture_generee, signe_le,
              adresse_client, ville_client, created_at
  partners  : id, user_id, nom, commission_montant, commission_payee, commission_pdf_url
  products  : données catalogue (source unique Supabase)
  site_content : CMS (logo, navigation, bannière, config pages)

Rôles :
  visitor | user | vip | partner | collaborateur | admin
  Source unique : client/src/utils/calculPrix.ts → calculerPrix()
  Multiplicateurs : user×1.5, partner×1.2, vip×1.3
