POINTS CLES POUR RECONSTRUCTION
================================================================

1. PRIX : Le prix d'achat est la source unique. Tout prix affiche
   est calcule dynamiquement selon le role (×2, ×1.2, ×1.3 ou negocie).

2. DEVIS PDF : 5 types de documents avec layout precis.
   L'emetteur est toujours LUXENT LIMITED (UK).
   TVA non applicable (art. 293 B du CGI).

3. PANIER : localStorage "rippa_cart", clear sur changement d'auth.

4. ROLES : admin → collaborateur → partner → vip → user → visitor
   Chaque role voit des prix differents et a des acces differents.

5. NUMEROTATION : Sequences SQL atomiques (D26XXXXX, FA26XXXXX, etc.)
   A reimplementer via Firestore counters ou Cloud Functions.

6. SEO : Conserver les URLs .html alternatives pour les liens existants.
   SPA avec rewrite universel vers index.html.

7. SHIPPING : Prix fixes par destination DOM-TOM, certaines "sur devis".

8. PARTENAIRES : Systeme de commission = prix negocie - prix partenaire.
